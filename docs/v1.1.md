# Lite ORM v1.1.0

v1.1.0 focuses on professional release readiness, diagnostics, examples, and cross-platform native packaging.

## New in v1.1.0

- `lite-orm doctor` CLI command for environment diagnostics.
- GitHub Actions CI matrix for Linux, macOS, Windows and Node 18/20/22/24.
- npm publish workflow with provenance support.
- Vendored SQLite amalgamation for more portable native builds.
- Runnable examples:
  - `examples/todo`
  - `examples/blog`
  - `examples/auth`
- Example test script: `npm run test:examples`.
- Package files now include `examples/`, `docs/`, and `deps/sqlite/`.

## Upgrade from 1.0.x

```bash
npm install @ghuts/liteorm@latest
```

The JavaScript API remains backward compatible with 1.0.x.

## Verify locally

```bash
npm run build
npm test
npm run test:types
npm run test:examples
node bin/lite-orm.js doctor
npm pack --dry-run
```

## Release checklist

```bash
git status --short
npm test
npm run test:examples
npm pack --dry-run
npm publish --dry-run --access public
```
