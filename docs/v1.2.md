# Lite ORM v1.2.0

v1.2.0 is the advanced native/TypeScript release.

## New in v1.2.0

- TypeScript-first model helpers: `InferModel`, `InferCreateInput`, `InferUpdateInput`, typed fields, hooks, JSON helpers, and policies.
- Paranoid soft deletes: `paranoid`, `deletedAt`, `withTrashed()`, `onlyTrashed()`, query restore, and force delete.
- Abortable lifecycle hooks with `ctx.abort()` and `HookAbortError`.
- Structured validation results via `validate()` / `assertValid()` and `ValidationError.issues`.
- Native prepared statements, statement cache statistics, JS UDFs, and custom collations.
- Nested transactions through savepoints, retry options, and transaction mode validation.
- JSON helper operators, JSON atomic updates, JSON expression indexes, and injection hardening.
- CDC/sync helpers, RBAC policies, query profiling, and encryption key rotation.
- FTS5 ranking, highlight/snippet helpers, rebuild/optimize, and trigger sync.
- CLI scaffolding plus migration status, preview, seed, and rollback commands.
- Optional sql.js adapter, playground example, cookbook, and benchmark script.

## Publish notes

The package remains scoped public under `@ghuts/liteorm` with:

```json
"publishConfig": {
  "access": "public"
}
```

Pre-publish command:

```bash
npm run publish:dry
```

Real publish command for the maintainer:

```bash
npm publish --access public
```
